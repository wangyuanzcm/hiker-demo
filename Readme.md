# demo

## 启动

```
    parcel src/index.html
```

## 实时监控

```
    parcel watch src/index.html
```

## 构建

```
    npm run build-xxx // 比如: npm run build-webTRC
```

3. 验证rrweb使用websocket实现直播模式，注意需要开启ws服务器

        先运行node server.js打开websocket服务，然后打开send和reception页面，点击send页面的开始录制即可
